# 🎵 Tkinter Music Data Registry App

Python desktop application built with Tkinter to manage personal data and musical preferences.

## 🚀 Features

* Register personal data (name, surname, DNI, city, country)
* Add musical preferences (song, artist, genre)
* Input validation (empty fields, alphabetic validation, DNI check)
* Data persistence using file storage
* Interactive table with scroll view
* Clean and structured GUI interface

## 🧠 Skills Demonstrated

* Python (Object-Oriented Programming)
* Multiple inheritance
* Tkinter GUI development
* Data validation and error handling
* File handling (read/write)
* Modular programming (separation of concerns)

## 🏗️ Project Structure

* `app.py` → Main application
* `filtro_datos.py` → Data validation module
* `registro_gustos_musicales.txt` → Data storage

## ▶️ How to Run

```bash
python app.py
```

## 🎯 About This Project

This project was built to practice building real-world desktop applications using Python, focusing on clean architecture, data validation and user interaction.
